Lemonade Stand
==============

Install and Run
---------------
*Requires python2.7*

* Clone this repository
* A `requirements.txt` file is included for easy python dependency installation.
    
```bash
# Install dependencies
$ pip install -r requirements.txt # Might require sudo if not in virtualenv
$ python app.py
```

---------------
